# ♻️ RecycleSmart - Projeto Acadêmico

O **RecycleSmart** é uma solução inteligente desenvolvida para facilitar o processo de reciclagem. O objetivo é permitir que usuários identifiquem materiais recicláveis através de fotos, texto ou comandos de voz.

## 🚀 Status do Projeto
Este repositório contém o **MVP (Mínimo Produto Viável)** inicial, focado na arquitetura básica de classes em Java.

## 🛠️ Tecnologias Utilizadas
*   **Linguagem:** Java 17+
*   **Ambiente:** VS Code
*   **Paradigma:** Orientação a Objetos (Encapsulamento e Tipagem Forte)

## 📋 Como funciona
Atualmente, o sistema possui:
- `RecyclableItem.java`: Modelo que representa o objeto a ser reciclado.
- `RecycleSmartApp.java`: Motor principal que processa a entrada e simula a classificação do material.

---
> *Nota: Este é um projeto para fins estritamente acadêmicos.*
